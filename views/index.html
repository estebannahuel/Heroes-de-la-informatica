La historia de la informática y la programación

Década de 1940: Los Inicios Surge la informática moderna con la construcción de las primeras computadoras electrónicas, como el ENIAC (1946), utilizadas principalmente para cálculos científicos y militares.

Década de 1950: Lenguajes de Programación Se desarrollan los primeros lenguajes de programación, como el Fortran (1957) y el COBOL (1959), permitiendo a los programadores escribir instrucciones en un formato más cercano al lenguaje humano.

Década de 1960: Sistemas Operativos Aparecen los primeros sistemas operativos, como el UNIX (1969), que facilitan la interacción entre el hardware y el software de las computadoras.

Década de 1970: Microprocesadores y Computadoras Personales, La invención del microprocesador (Intel 4004, 1971) allana el camino para las computadoras personales, como el Altair 8800 (1975) y el Apple I (1976), que desencadenan la revolución de la informática doméstica.

Década de 1980: GUI y Computadoras Personales Se popularizan las interfaces gráficas de usuario (GUI) con computadoras como el Apple Macintosh (1984) y el sistema operativo Windows (1985), facilitando la interacción con las computadoras.

Década de 1990: Internet y la World Wide Web La invención de la World Wide Web (1990) por Tim Berners-Lee y el crecimiento de Internet transforman la forma en que se comparte la información y se comunican las personas.
